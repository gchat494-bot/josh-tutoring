What Is a System of Two Linear Equations?

Before solving, you need to understand what a system actually is — and the three possible outcomes when two lines meet (or don't) on a graph.

The Core Idea

A system of two linear equations is a pair of equations with two unknowns — usually x and y. A solution is a single ordered pair (x, y) that makes both equations true at the same time. Graphically, it's the point where the two lines intersect.

A system looks like this:

ax + by = c
dx + ey = f

where a, b, c, d, e, and f are constants. Your job is to find the (x, y) pair that satisfies both equations simultaneously.

Three Possible Outcomes
One Solution
Lines cross at exactly one point
No Solution
Parallel lines — never meet
Infinitely Many
Same line — overlap completely
Outcome Graphical Meaning How to Identify
One solution Lines intersect at exactly one point Different slopes (m values)
No solution Lines are parallel — never intersect Same slope, different y-intercepts
Infinitely many Lines are the same line Same slope AND same y-intercept

SAT tip: To quickly identify the number of solutions, rewrite both equations in slope-intercept form y = mx + b and compare slopes and intercepts. This is one of the most tested concepts on the SAT — expect at least 1–2 questions per test!
