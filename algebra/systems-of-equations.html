Systems of Two Linear Equations

Learn how to solve a pair of linear equations simultaneously using substitution and elimination — and how to recognize when a system has no solution or infinitely many solutions.

Core Concept

A system of two linear equations has two unknowns, usually x and y. The solution is an ordered pair (x, y) that satisfies both equations simultaneously — graphically, it's the point where the two lines intersect.

Number of Solutions
Outcome Graphical Meaning Algebraic Sign
One solutionLines intersect at one pointDifferent slopes
No solutionLines are parallelSame slope, different y-intercepts
Infinitely many Lines are identicalSame slope and y-intercept

SAT tip: Rewrite both equations as y = mx + b. Same slope, different intercept = no solution. Same slope, same intercept = infinitely many solutions. Different slopes = exactly one solution.

Solving Methods

Method 1: Substitution

Best when one variable is already isolated (y = ... or x = ...)
  1. Isolate one variable in one equation
  2. Substitute into the other equation
  3. Solve for the remaining variable
  4. Back-substitute to find the other

Method 2: Elimination

Best when both equations are in Ax + By = C form
  1. Multiply to make coefficients opposites
  2. Add equations to cancel one variable
  3. Solve for the remaining variable
  4. Back-substitute to find the other
Worked Example — Substitution
y = 2x + 1
3x + y = 16
1
y is already isolated: y = 2x + 1
2
Substitute into second equation: 3x + (2x + 1) = 16
3
Simplify: 5x + 1 = 16 → 5x = 15 → x = 3
4
Back-substitute: y = 2(3) + 1 = 7
✓
Answer: (x, y) = (3, 7)
Worked Example — Elimination
2x + 3y = 12
4x − 3y = 6
1
Coefficients of y are already opposites (+3y and −3y)
2
Add equations: 6x = 18
3
Solve: x = 3
4
Back-substitute: 2(3) + 3y = 12 → y = 2
✓
Answer: (x, y) = (3, 2)

SAT shortcut: If the problem only asks for a combination like x + y or 2x − y, try adding or subtracting the original equations directly. You can often get the answer in one step without solving for x and y individually!
